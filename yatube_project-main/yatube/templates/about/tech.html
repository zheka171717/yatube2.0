Стек технологий

Как создавался проект

Проект Yatube создавался пошагово, следуя урокам по Django:

  • Создана структура проекта и настройки окружения
  • Разработаны модели Post и Group для хранения постов и сообществ
  • Реализованы view-функции и шаблоны для отображения контента
  • Добавлена пагинация для удобной навигации
  • Создана система регистрации и авторизации пользователей
  • Реализован поиск по постам с подсветкой результатов
  • Добавлены страницы профайлов пользователей и детального просмотра постов
